# Update

## Query

```plain
PATCH /api/v0/organisation/{ORG_ID}
```

- `ORG_ID`: Identifier or name of the organisation

## Request Body Example

```json
{
    "description": "SOC level 1 team",
    "name": "soc-level1"
}
```

- `name` is editable
- `description` is editable
